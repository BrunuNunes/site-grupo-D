o que é GIT?

GIT é um sistema de controle de versão de arquivos que registra as mudanças feitas em arquivos ou um conjunto periodicamente, de maneira que possamos recupera-las. Este sistema possui uma plataforma que possibilita ao usuário voltar a versão anterior, não permitindo que o usuário faça o upload de arquivos corrompidos.
